Commande, paiement et verification

Tout est guide ici pour eviter les erreurs: commande envoyee, validation manager, paiement, verification puis QR code ou livraison.

Etapes du parcours
  1. Le manager verifie la commande et confirme qu'elle peut etre preparee.
  2. Le client paie exactement le montant affiche.
  3. Le staff controle la preuve avant de valider le paiement.
  4. Un QR code ou une livraison est ensuite debloque(e).
